# Examples

This page collects copy-pasteable starting points. The Atomic repo also includes runnable examples under `examples/` and package-specific example directories.

## Research before editing

```text
Use the research-codebase skill to understand how API authentication works. Write a dated research doc under research/docs/. Do not edit production code.
```

## Focused bug fix

```text
Reproduce the failing test in test/auth/session.test.ts. Identify the root cause before editing. Make the smallest safe fix, then run that test again. Return changed files and command results.
```

## Review a staged diff

```bash
git diff --staged | atomic -p "Review this staged diff for correctness, security, and test coverage. Return blockers first, then optional improvements."
```

## Create a spec from research

```text
Use the create-spec skill to write an implementation spec for migrating auth middleware to session-v2. Ground it in research/docs/2026-05-11-auth-middleware.md and include validation steps.
```

## Scaffold a workflow

```text
Use the workflow skill to create a project workflow named issue-to-pr. It should accept an issue number, fetch the issue, research affected files, pause for approval, implement, run tests, and produce a PR summary.
```

## Add a custom tool through an extension

```ts
import type { PiExtension } from "@bastani/atomic";

const extension: PiExtension = (pi) => {
  pi.registerTool({
    name: "project_status",
    description: "Return project-specific status information.",
    parameters: {
      type: "object",
      properties: {},
    },
    async execute() {
      return { content: "Project status: ok" };
    },
  });
};

export default extension;
```

Place project extensions under `.atomic/extensions/` and see [Extensions](/customization/extensions) for the full API.

## Minimal SDK script

```ts
import { createAgentSession } from "@bastani/atomic";

const session = await createAgentSession({
  cwd: process.cwd(),
  model: "anthropic/claude-sonnet-4",
});

const result = await session.prompt("Summarize this repository in 10 bullets.");
console.log(result.text);
```

See [SDK](/reference/sdk) for current APIs and options.

## Safe autonomous workflow prompt

```text
Run the ralph workflow for this bounded task: implement the validation behavior described in specs/input-validation.md. Stop and ask before changing public API shape. Validate with bun test test/input-validation.test.ts and summarize any remaining risks.
```
